# Mothers of Hind — Website

Non-profit charity website (UK / Manchester) rebuilding education, mental-health
care and dignity for children and families in Gaza and other communities
affected by war — in memory of Hind Rajab.

## Stack
- Front-end: vanilla HTML/CSS/JS
- Back-end: Node.js + Express API for Instagram feed

## Running locally
```bash
npm install
cp .env.example .env
npm start
```
